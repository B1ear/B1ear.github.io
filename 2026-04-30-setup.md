### 2. `_config.redefine.yml` (主题配置)

新增以下配置：

```yaml
# 基本信息
info:
  title: Bear
  subtitle: ""
  author: B1ear
  url: https://B1ear.github.io

# 首页横幅
home_banner:
  enable: true
  style: fixed
  image:
    light: /images/bg.jpg
    dark: /images/bg.jpg
  title: Bear
  subtitle:
    text:
      - I'm fine thank you
  text_color:
    light: "#fff"
    dark: "#d1d1b6"

# 导航栏（透明背景）
navbar:
  color:
    left: "transparent"
    right: "transparent"
    transparency: 10

# 页脚
footer:
  runtime: true
  start: 2026/04/30 00:00:00
  statistics: true

# 自定义注入
inject:
  enable: true
  head:
    - "<style>.home-banner-background{filter:blur(5px)}.navbar-content.has-home-banner{--first-text-color:#fffde7;--default-text-color:#fffde7}</style>"
    - "<script>document.addEventListener('DOMContentLoaded',function(){document.title=document.title.replace(/ - .*/,'')})</script>"

# 插件
plugins:
  mathjax:
    enable: true
```

### 3. `source/images/bg.jpg`

## 自定义样式说明

| 功能 | 实现方式 |

| 导航栏文字淡黄色 | CSS 变量 `--first-text-color: #fffde7` |
| 去掉页面标题副标题 | JavaScript 正则替换 |
